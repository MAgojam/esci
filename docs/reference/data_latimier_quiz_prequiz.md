# Latimier Quiz Prequiz - Ch07 - Quiz and Prequiz groups in Latimier et al. (2019)

Just the Quiz (RQ) an Prequiz (QR) groups from Latimier et al., 2019 See
full details in data_latimier_3_groups

## Usage

``` r
data_latimier_quiz_prequiz
```

## Format

### `data_latimier_quiz_prequiz`

A data frame with 190 rows and 3 columns:

- Group:

  factor

- Test%:

  numeric

- Time:

  numeric
