# Sleep Beauty - Ch11 - for End-of-Chapter Exercise 11.6

Is there really such a thing as beauty sleep? To investigate,
researchers decided to examine the extent to which sleep relates to
attractiveness. Each of 70 college students self-reported the amount of
sleep they had the night before. In addition, a photograph was taken of
each participant and rated for attractiveness on a scale from 1 to 10 by
two judges of the opposite gender. The average rating score was used.
You can download this data set (Sleep Beauty) from the book website.

## Usage

``` r
data_sleep_beauty
```

## Format

### `data_sleep_beauty`

A data frame with 70 rows and 2 columns:

- Sleep (hours):

  numeric

- Rated_Attractiveness:

  numeric
