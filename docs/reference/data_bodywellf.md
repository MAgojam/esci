# BodyWellF - Ch12 - Body Satisfaction and Well-being data for females from Figure 11.24 right panel

A subset of data_bodywell_fm, reports only for those participants who
identified as female. Data is Subjective Wellbeing and Body
Satisfaction.

## Usage

``` r
data_bodywellf
```

## Format

### `data_bodywellf`

A data frame with 59 rows and 2 columns:

- Body Satisfaction:

  numeric

- Well-being:

  numeric
