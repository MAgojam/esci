# BodyWellM - Ch12 - Body Satisfaction and Well-being data for males from Figure 11.24 left panel

A subset of data_bodywell_fm, reports only for those participants who
identified as male. Data is Subjective Wellbeing abd Body Satisfaction.

## Usage

``` r
data_bodywellm
```

## Format

### `data_bodywellm`

A data frame with 47 rows and 2 columns:

- Body Satisfaction:

  numeric

- Well-being:

  numeric
