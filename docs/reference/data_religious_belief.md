# Religious belief - Ch03 - for End-of-Chapter Exercise 3.5

Let's look at some data about religious beliefs. The Religious belief
file has data from a large online survey in which participants were
asked to report, on a scale from 0 to 100, their belief in the existence
of God. Age was also reported.

## Usage

``` r
data_religious_belief
```

## Format

### `data_religious_belief`

A data frame with 213 rows and 3 columns:

- Response_ID:

  character

- Belief_in_God:

  factor

- Age:

  integer
