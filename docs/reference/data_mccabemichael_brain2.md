# McCabeMichael brain2 - Ch9 - from Michael et al. (2013)

Same as data_mccabemichael_brain but includes two additional critique
studies run by the Michael team.

## Usage

``` r
data_mccabemichael_brain2
```

## Format

### `data_mccabemichael_brain2`

A data frame with 12 rows and 9 columns:

- Study name:

  factor

- M No Brain:

  numeric

- s No Brain:

  numeric

- n No Brain:

  numeric

- M Brain:

  numeric

- s Brain:

  numeric

- n Brain:

  numeric

- SimpleCritique:

  factor

- Research group:

  factor
