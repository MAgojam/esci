# Generates a forest plot displaying results of a meta-analysis

\`plot_meta' returns a ggplot2 object visualizing the results of a
meta-analysis, showing each study effect size and CI, the overall effect
size and CI as a diamond, effect sizes estimated at each moderator level
(if defined), and (optionally) prediction intervals for subsequent
studies. This function requires as input an esci_estimate object
generated by an esci meta-analysis function:
[`meta_any()`](https://rcalinjageman.github.io/esci/reference/meta_any.md),
[`meta_d1()`](https://rcalinjageman.github.io/esci/reference/meta_d1.md),
[`meta_d2()`](https://rcalinjageman.github.io/esci/reference/meta_d2.md),
[`meta_mdiff_two()`](https://rcalinjageman.github.io/esci/reference/meta_mdiff_two.md),
[`meta_mean()`](https://rcalinjageman.github.io/esci/reference/meta_mean.md),
[`meta_pdiff_two()`](https://rcalinjageman.github.io/esci/reference/meta_pdiff_two.md),
[`meta_proportion()`](https://rcalinjageman.github.io/esci/reference/meta_proportion.md),
and
[`meta_r()`](https://rcalinjageman.github.io/esci/reference/meta_r.md).

## Usage

``` r
plot_meta(
  estimate,
  mark_zero = TRUE,
  include_PIs = FALSE,
  report_CIs = FALSE,
  explain_DR = FALSE,
  meta_diamond_height = 0.35,
  ggtheme = ggplot2::theme_classic()
)
```

## Arguments

- estimate:

  - an esci_estimate object generated by an esci meta\_ function

- mark_zero:

  - Boolean; defaults to TRUE to include a dotted line indicated no
    effect (effect_size = 0)

- include_PIs:

  - Boolean; defaults to FALSE; set to TRUE to include prediction
    intervals for the overall effect and each moderator level (if
    defined)

- report_CIs:

  - Boolean; defaults to FALSE; set to TRUE to include printed
    representation of each study effect size and CI along the right-
    hand of the figure

- explain_DR:

  - Boolean; defaults to FALSE; set to TRUE if no moderator is defined
    to show both the RE and FE effect sizes to represent how the diamond
    ration measure of effect-size heterogeneity is calculated

- meta_diamond_height:

  - Optional real number \> 0 to indicate height that each meta-analytic
    diamond should be drawn; defaults to 0.35

- ggtheme:

  - Optional ggplot2 theme object to control overall styling; defaults
    to
    [`ggplot2::theme_classic()`](https://ggplot2.tidyverse.org/reference/ggtheme.html)

## Value

Returns a ggplot object

  comparison_sds = "s Brain",
  comparison_ns = "n Brain",
