# SmithRecall - Ch15 - from Smith et al. (2016)

SmithRecall - Ch15 - from Smith et al. (2016)

## Usage

``` r
data_smithrecall
```

## Format

### `data_smithrecall`

A data frame with 120 rows and 6 columns:

- ID:

  integer

- Study Technique:

  factor

- Stress Status:

  factor

- %Recalled:

  numeric

- Items Recalled:

  numeric

- Group:

  factor
